## Overview

This repository documents a **60-Day Python Challenge** — a structured, self-paced learning path covering Python from the very fundamentals through to advanced, object-oriented concepts. The goal was threefold: build a disciplined, day-by-day learning habit; strengthen core Python foundations ahead of applying them to data science and machine learning work; and keep a public, well-documented record of every concept and mini-project along the way.

The challenge wraps up with a capstone project — a console-based **Mini Banking System** — that pulls file handling, OOP, and functional-programming concepts from all three phases together into one complete application.
