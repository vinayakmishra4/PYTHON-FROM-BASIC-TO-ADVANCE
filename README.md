## Overview

This repository documents a **60-Day Python Challenge** — a structured, self-paced learning path covering Python from the very fundamentals through to advanced, object-oriented concepts. I took on this challenge to build a disciplined, day-by-day learning habit, strengthen my core Python foundations ahead of applying them to data science and machine learning work, and keep a public, well-documented record of every concept and mini-project along the way.

## Course Info

| Detail | Description |
|---|---|
| **Format / Provider** | Self-paced, independent study (personal challenge) |
| **Duration** | 60+ Days — originally 3 phases of 20 days, now extended with a 4th phase |
| **Completion Status** | ✅ Phases 1–3 complete (Day 1–60) · 🚧 Phase 4 in progress (Day 61 onward) |
| **Prerequisites** | None — designed to start from absolute basics |

## Syntax / Key Concepts

General patterns practiced throughout the challenge (conceptual only, not runnable code):

| Concept | General Pattern |
|---|---|
| Variable assignment | `variable_name = value` |
| Conditional logic | `if condition: ... elif condition: ... else: ...` |
| Iteration | `for item in sequence: ...` and `while condition: ...` |
| Function definition | `def function_name(parameters): ...` |
| Exception handling | `try: ... except ExceptionType: ... finally: ...` |
| Class definition | `class ClassName: def __init__(self, ...): ...` |
