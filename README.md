# ⚡ Quick Start

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/PYTHON-FROM-BASIC-TO-ADVANCE.git
```

## 2. Open Folder

```bash
cd PYTHON-FROM-BASIC-TO-ADVANCE
```

## 3. Install Python

Download Python 3.8 or later from:

https://www.python.org/downloads/

## 4. Start Learning

Open **Day 1** and continue one lesson every day.
