## 🚀 Getting Started
1. Install Python 3.8+ from [python.org](https://www.python.org/downloads/)
2. Install a code editor (VS Code or PyCharm recommended)
3. Open your terminal and verify Python with:
   ```bash
   python --version
   ```
4. You're ready to start Day 1!
